# Contributing

Thanks for your interest in podblock!

## Getting Started

1. Fork the repo
2. Clone your fork
3. `make build` to compile
4. `make run` to test locally

## Making Changes

- Keep it simple — this is a single-file project and should stay that way
- Test with real audio devices before submitting (connect/disconnect AirPods, etc.)
- No external dependencies — stdlib + CoreAudio only

## Submitting a PR

1. Create a branch from `main`
2. Make your changes
3. Verify `make build` compiles without warnings
4. Open a PR with a clear description of what changed and why
